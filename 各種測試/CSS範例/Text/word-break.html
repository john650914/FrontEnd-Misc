這跟word-wrap好像有點不一樣，word-break會把每一個英文單字全部切斷立刻換行，所以每行未都是切齊的；而word-wrap只會把空間放不下的超長字做斷字，閱讀上比較自然。

word-break:

The word-break property specifies line breaking rules for non-CJK scripts.

Tip: CJK scripts are Chinese, Japanese and Korean ("CJK") scripts.

Note: The word-break property is not supported in Firefox or Opera.
