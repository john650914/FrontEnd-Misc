font：

用于指定或接收最多六个以下空格分隔属性值的字符串，或者是用于设置六种列出的文本字体的字符串：可以按顺序设置如下属性：

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

此屬性是複合屬性，最多可以指定6個值。font-style、font-variant 和 font-weight 值可按任何順序出現在 font-size 之前。但是，font-size、line-height 和 font-family 屬性必須按所列順序顯示。在設置 font 屬性時，同時會設置元件屬性。在這種情況下，字串必須是元件屬性的有效值的組合；只有 font-family 值才能具有多個值。如果字串不包含某個元件屬性的值，則該屬性將設置為其預設值，而不管該元件屬性以前的設置如何。由於有一些字體本身沒有斜體樣式,所以有時斜體顯示將被忽略。

範例1，這一段文字樣式設定為：Style:oblique，Variant:small-caps，Weight:bold，Size:16px，Height:28px，Family:Arial,Times。
範例2，也可以省略部份屬性值，例如這一段文字只有設定font-ize和font-family。
