HTML特殊又少用的標籤：ruby, rb, rt, rp, bdo, nobr, wbr,marquee,blink, multicol

學習html時，曾經發現了幾個非常少見又有點特殊的標籤元素，意思是...看看就好，基本上都用不太到。

Ruby 注音標示

此標籤元素屬於xhtml 1.1版本的一部分，但尚未廣泛使用，因此可能會有瀏覽器不支援的情形。但主要用於中文與日文的注音標示，必須配合rb, rt, rp三個子元素。需要起始標籤與結尾標籤。

屬性 屬性值 預設值 說明
name 字串 設定元素名稱。
tabindex 0~32767 設定tab鑑的移動順序。無障礙網頁常使用屬性之一。
accesskey 任何一個字元 設定快速鍵文字，當系統為windows時，則為alt+設定字元；如果是mac系統則為ctrl+設定字元。
language 字串 設定文字語言。
通用屬性：id, class, style, title, dir, lang, xml:lang

Rb 注音標示對象

設定被標示的元素對象，為ruby的子元素。可用於ruby以外的行內元素(例如：span)

屬性 屬性值 預設值 說明
name 字串 設定元素名稱。
tabindex 0~32767 設定tab鑑的移動順序。無障礙網頁常使用屬性之一。
accesskey 任何一個字元 設定快速鍵文字，當系統為windows時，則為alt+設定字元；如果是mac系統則為ctrl+設定字元。
language 字串 設定文字語言。
通用屬性：id, class, style, title, dir, lang, xml:lang

Rt 標示文字

設定顯示的注音文字，為ruby的子元素。可用於ruby以外的行內元素(例如：span)

屬性 屬性值 預設值 說明
name 字串 設定元素名稱。
tabindex 0~32767 設定tab鑑的移動順序。無障礙網頁常使用屬性之一。
accesskey 任何一個字元 設定快速鍵文字，當系統為windows時，則為alt+設定字元；如果是mac系統則為ctrl+設定字元。
language 字串 設定文字語言。
rbspan 數值 設定rt所對應的rb數量，必須為正整數，最小單位為1。
通用屬性：id, class, style, title, dir, lang, xml:lang

Rp 顯示無法對應的符號

設定無法顯示的符號，例如括號等等，當瀏覽器不支援ruby等元素時，可能會被當作文字顯示，此時rp用來顯示開頭與結尾，若瀏覽器支援ruby等元素，則不顯示rp元素。rp為ruby的子元素。只能用於ruby元素內。

屬性 屬性值 預設值 說明
通用屬性：id, class, style, title, dir, lang, xml:lang

    範例26-1：ruby, rb, rt, rp 基本範例

    新年快樂 ( ㄒㄧㄣ ㄋㄧㄢˊ ㄎㄨㄞˋ ㄌㄜˋ )

  • 語法範例：<ruby>
    <rb>新年快樂</rb>
    <rp>(</rp>
    <rt rbspan="1">ㄒㄧㄣ ㄋㄧㄢˊ ㄎㄨㄞˋ ㄌㄜˋ</rt>
    <rp>)</rp>
    </ruby>

    測試結果：只有ie7可正常顯示！(所以...真的看看就好)

bdo 改變文字方向

用於改變文字方向，利用unicode雙向算法根據字元排列順序做改變。必須要有起始標籤與結尾標籤。

屬性 屬性值 預設值 說明
dir ltr/rtl 設定文字方向，ltr為從左至右，rtl為由右至左。
通用屬性：id, class, style, title, dir, lang, xml:lang

    範例27-1：bdo 基本範例

    由左至右：新年快樂。
    由右至左：新年快樂。

  • 語法範例：由左至右：<bdo dir="ltr">新年快樂</bdo>。
    由右至左：<bdo dir="rtl">新年快樂</bdo>。

    特殊設計專用？

nobr 禁止斷行

設定文字不得自動換行。不過已不建議使用，理由是nobr元素並沒有被W3C列入規範，建議改以CSS樣式：white-space:nowarp來取代。

屬性 屬性值 預設值 說明
通用屬性：id, class, dir, lang, xml:lang

wbr 設定斷行處

設定文字在某處設定斷行，通常與nobr搭配使用。跟br元素其實差不多，只是不需要搭配nobr，感覺使用nobr與wbr...好像搞的很勞神一樣，且wbr元素並沒有被W3C列入規範，不建議使用，所以使用br即可。

使用方式：<wbr id=''屬性值/>

屬性 屬性值 預設值 說明
通用屬性：id

marquee 跑馬燈

設定文字或是圖片等製作跑馬燈效果，並且可搭配behavior、direction屬性設定來顯示不同效果，但marquee元素並沒有被W3C列入規範，不建議使用。必須使用起始標籤與結尾標籤。

屬性 屬性值 預設值 說明
behavior scroll/slide/alternate 設定跑馬燈效果，分別為：捲動、滑動以及交替。
direction left/right/up/down 設定跑馬燈移動方向。
scrollamount 數值 設定移動距離，數值必須為正整數。
scrolldelay 毫秒 設定效果延遲時間。
truespeed 毫秒 設定跑馬燈移動速度。
loop 數值 設定跑馬燈執行次數，如為-1時則為無限循環。
bgclolor 顏色值 設定跑馬燈背景顏色。
height 數值/百分比 設定跑馬燈區域高度，數值必須為正整數，如果使用百分比則必須加上「%」符號。
width 數值/百分比 設定跑馬燈區域寬度，數值必須為正整數，如果使用百分比則必須加上「%」符號。
hspace 數值 設定跑馬燈區域左右預留空間，數值必須為正整數。
vspace 數值 設定跑馬燈區域上下預留空間，數值必須為正整數。
tabindex 0~32767 設定tab鍵時移動順序。
accesskey 任何一個字元 設定快速鍵文字，如果系統為windows時，則為atl+設定字元；如果系統為mac時，則為ctrl+設定字元。
language 字串 設定跑馬燈的語言碼。
通用屬性：id, class, style, title, dir, lang, xml:lang

    範例28-1：marquee 基本範例

    祝大家新年快樂！

  • 語法範例：<marquee behavior="slide" bgcolor="#1a1a1a" direction="left" hspace="10" vspace="15" width="680" height="18" loop="-1" truespeed="100" scrollamount="10" scrolldelay="100" style="color:#9C0;">祝大家新年快樂！</marquee>

blink 閃爍文字

設定文字閃爍效果，blink元素並沒有被W3C列入規範，不建議使用。必須使用起始標籤與結尾標籤。

屬性 屬性值 預設值 說明
通用屬性：id, class, style, title, dir, lang, xml:lang

    範例29-1：blink 基本範例

    新年快樂

  • 語法範例：<blink>新年快樂</blink>

multicol 多重欄位

設定內容產生分欄方式排列，成多欄顯示。multicol元素並沒有被W3C列入規範，不建議使用。必須使用起始標籤與結尾標籤。

屬性 屬性值 預設值 說明
cols 數值 設定分割欄位數，數值必須為正整數，且為必要屬性，不得小於1。
gutter 數值 10 設定欄位之間的距離，數值必須為正整數。
width 數值 設定整體寬度，數值必須為正整數。
通用屬性：id, class, style, title, dir, lang, xml:lang
