@keyframes:

參考：https://msdn.microsoft.com/library/jj680076(v=vs.85).aspx

With the @keyframes rule, you can create animations.

The animation is created by gradually changing from one set of CSS styles to another.

During the animation, you can change the set of CSS styles many times.

Specify when the change will happen in percent, or the keywords "from" and "to", which is the same as 0% and 100%.

0% is the beginning of the animation, 100% is when the animation is complete.

For best browser support, you should always define both the 0% and the 100% selectors.

Note: Use the animation properties to control the appearance of the animation, and also to bind the animation to selectors.

語法：@keyframes animationName{keyframes-selector1{css-styles} keyframes-selector2{css-styles} ...};

這個屬性目前只有google chrome、safari及Firefox有支援。

Property Description Value
animationName Required. Defines the name of the animation. Any valid name of css rule.
keyframes-selector Required. Percentage of the animation duration.
Note: You can have many keyframes-selectors in one animation.
 0-100%

or

from (same as 0%)
to (same as 100%)
css-styles Required. One or more legal CSS style properties css
