animation-iteration-count:

The animation-iteration-count property defines how many times an animation should be played.

語法：animation-iteration-count:number|infinite

這個屬性目前只有google chrome、safari及Firefox有支援。

Property Description Value
animation-name Specifies the name of the keyframe you want to bind to the selector keyframename | none
animation-duration Specifies how many seconds or milliseconds an animation takes to complete time
animation-timing-function Specifies the speed curve of the animation linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n)
animation-delay Specifies a delay before the animation will start time
animation-iteration-count Specifies how many times an animation should be played number | infinite
animation-direction Specifies whether or not the animation should play in reverse on alternate cycles normal | alternate
animation-play-state Specifies whether the animation is running or paused. paused | running
