target-new:

设置新创建的是窗口还是标签还是不创建。

語法：target-new: window|tab|none;

Note: The target-new property only works if the target-name property creates a new tab or a new window.

示範：Open new windows in new tabs instead of new windows.

a{target-name:new; target-new:tab;}

windowOpen link in new window

tabOpen link in a new tab of an existing window

noneNo new destination is created

這個屬性目前沒有任何瀏覽器支援。

Property Description
target A shorthand property for setting the target-name, target-new, and target-position properties.
target-name Specifies where to open hyperlinks (target destination)
target-new Specifies whether new destination links should open in a new window or in a new tab of an existing window
target-position Specifies where new destination links should be placed
